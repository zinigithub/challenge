Instrucciones para la ejecución:

El script ha sido probado con python3

python3 process.py -archivojson data_json.txt -archivocsv datos_csv.csv


Librerias necesarias:
sendmail (sudo apt-get install sendmail)
python3 sudo apt-get install python3

Es necesario instalar sendmail en la maquina donde correrá el script para el envío de correo electrónico, sin esto el script no funcionará correctamente.
